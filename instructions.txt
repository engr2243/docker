Use containers for development: Example
A) Docker build technique:
A.1) Run a database in a container
1) docker volume create mysql
2) docker volume create mysql_config
3) docker network create mysqlnet
4) sudo service mysql stop : Note: If mysql is already running stop to free port range 3306:3306

5) docker run --rm -d -v mysql:/var/lib/mysql \
  -v mysql_config:/etc/mysql -p 3306:3306 \
  --network mysqlnet \
  --name mysqldb \
  -e MYSQL_ROOT_PASSWORD=p@ssw0rd1 \
  mysql

6) docker exec -ti mysqldb mysql -u root -p

A.2) Connecting application with container.
1) Script for application app.py
2) pip3 install mysql-connector-python
3) pip3 freeze  >> requirements.txt
4) Maker docker file.
4) docker build --tag python-docker-dev .
5) docker run \
  --rm -d \
  --network mysqlnet \
  --name rest-server \
  -p 8000:5000 \
  python-docker-dev
6) curl http://localhost:8000/initdb, curl http://localhost:8000/widgets
